# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in react-keyboard-shortcuts, please email [amarkanala@gmail.com](mailto:amarkanala@gmail.com) instead of using the issue tracker.

**Please do not publicly disclose the vulnerability until it has been addressed.**

When reporting a vulnerability, please include:

- Description of the vulnerability
- Steps to reproduce the issue
- Potential impact
- Any suggested fixes (if available)

## Security Best Practices

When using react-keyboard-shortcuts:

- Keep your React version up to date
- Update react-keyboard-shortcuts regularly
- Use HTTPS when transmitting sensitive data
- Never hardcode sensitive information in your code
- Review keyboard shortcut handlers for security implications
